Welcome to my portfolio

My ongoing and completed projects covering VHDL, MIPS, VBA and MATLAB.

IVPRO

An open-source real-time image and video processing framework for FPGA applications written in VHDL.

AIS Latency Detector

A tool for testing the latency of AIS transponders, by reading specific NMEA sentences over serial lines.

RISC Matrices Multiplication

A low level tool for multiplying matrices with one another, designed for MIPS architecure processors.

FPGA Based MODEM

A MODEM written in VHDL that can generate random or predefined values, perform QPSK modulation, simulate a tranmission channel, induce an adjustable error and demodulate the signal.

VBA Email Filter

A VBA script used to prevent emails going to the wrong people.

FIR and IIR MATLAB Filters

Filter analysis and design using MATLAB.

Raspberry PI OCR

Optical character recognition using a Raspberry PI Zero H board.
